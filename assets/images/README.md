# Profile Images

## Profile Photo
Place your profile photo here as `profile.jpg`

**Requirements:**
- Filename: `profile.jpg`
- Format: JPG, PNG, or WebP
- Recommended size: 400x400px (square aspect ratio)
- File size: Keep under 500KB for optimal performance
- Quality: High resolution for professional appearance

## How to Add Your Photo
1. Take or select a professional headshot photo
2. Crop it to a square aspect ratio (1:1)
3. Resize to approximately 400x400 pixels
4. Save as `profile.jpg` in this directory
5. The image will automatically appear on the welcome page

## Tips for Best Results
- Use good lighting and a clean background
- Wear professional attire
- Smile naturally
- Ensure the photo is clear and in focus
- Consider using a professional photographer for best results
